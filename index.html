Generic Connection Framework by Marcel Patzlaff

Generic Connection Framework for Java SE

This project aims to provide a stable implementation of the Generic Connection Framework for the Java Standard Edition. It is not considered to be an implementation of any GCF-related JSR. Instead those are just used as an orientation.

Dependencies

  • GCF-Bluetooth (Bluecove) depends and includes the Bluecove library
  • GCF-Comm (RXTX) depends on the RXTX library
